# 📖 Project Overview

**ERROR_NEST** is a modern **Centralized Bug Management System** developed as a **Final Year Major Project**.

The application helps software development teams efficiently report, assign, monitor, prioritize, and resolve software bugs from one centralized platform.

It enables seamless collaboration between **Admins**, **Project Managers**, **Developers**, and **Testers**, improving communication, reducing bug resolution time, and enhancing software quality.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/John9899CSE/ERROR_NEST.git
```

---

## Navigate

```bash
cd ERROR_NEST
```

---

## Install Dependencies

```bash
npm install
```

---

## Run Development Server

```bash
npm run dev
```

---

Open your browser:

```
http://localhost:5173
```
